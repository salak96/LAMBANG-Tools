# 🚀 LAMBANG Tools AI

Platform **AI Tools & Affiliate Content Generator** untuk membantu membuat konten video, prompt AI, dan tools downloader dalam satu dashboard.

Website ini berisi kumpulan tools seperti:

* 🎬 AI Video Generator
* 🤖 Prompt AI Tools
* 🎥 Sora Video Downloader
* 🧠 Affiliate Content Generator
* 🔧 AI Utilities

# 📦 Installation

Clone repository:

```bash
git clone https://github.com/salak96/LAMBANG-Tools.git
```

Masuk ke folder project:

```bash
cd LAMBANG-Tools
```

Install dependencies:

```bash
npm install
```

Run development server:

```bash
npm run dev
```

Website akan berjalan di:

```
http://localhost:5173
```

---

# 🏗 Build Project

Untuk build production:

```bash
npm run build
```

Preview build:

```bash
npm run preview
```
